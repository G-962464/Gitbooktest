第一节

服务器相关知识

  1. Tomcat 搭建文件服务器

    • 方法1：
      1. 修改conf/web.xml文件
      2. 修改conf/server文件

    • 方法2：

      1. 同上
      2. 在tomcat安装目录\conf\Catalina\localhost下建立任意文件名（url访问路径）xml文件(多个既多个访问路径)，比如：download.xml，内容如下：
      <?xml version="1.0" encoding="UTF-8"?>  
<Context  path="/do" docBase="D://download" crossContext="true"></Context>
//path为url访问路径 docbase为绝对路径

  1. Nginx 修改主页

    • 方法1：

      nginx root /usr/share/nginx/html/index.html

    • 方法2：

      修改/etc/nginx/conf.d/nginx.conf

  1. Centos 系统安装Nginx

    cat /etc/redhat-release

    sudo rpm -ivh http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm

    yum repolist

    yum -y install nginx

    systemctl enable nginx

    systemctl start nginx

    systemctl stop firewalld

    curl -i localhost
